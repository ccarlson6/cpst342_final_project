Image Optimization Settings
---------------------------------------------

Thumbnails optimized at 80%-90% reduction. Keep file size at approx 100KB.
File size files optimized at 30%-50% reduction. Keep file size at approx 200KB-300KB.

Naming convention for photo album images:
	- thumbnails:        countrycode-location-photo#-min.jpg
	- full size images:  countrycode-location-photo#.jpg
